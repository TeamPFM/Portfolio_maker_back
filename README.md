# 사용 기술

## Backend: Nest.js (9.0.0)

- Database: MySql (3.1.2)

- DB Validation: Joi (17.7.1)

- ORM: TypeORM (0.3.12)

- Login: Json Web Token (passport-jwt 3.0.8)

- Encrypt: Bcrypt (5.1.0)

- Logger: Winston logger (3.8.2)

- Security: Helmet (6.0.1)

## Frontend: React.js
